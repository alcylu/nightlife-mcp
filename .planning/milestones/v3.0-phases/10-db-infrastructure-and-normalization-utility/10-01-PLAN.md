  <files>supabase/migrations/20260312_fuzzy_search.sql, supabase/migrations/20260312_fuzzy_search_index.sql</files>
  <action>
Create two SQL migration files:

**File 1: `supabase/migrations/20260312_fuzzy_search.sql`** (transactional — extensions + wrapper + RPC)

1. Enable extensions (idempotent):
   ```sql
   CREATE EXTENSION IF NOT EXISTS unaccent;
   CREATE EXTENSION IF NOT EXISTS pg_trgm;
   ```

2. Create IMMUTABLE wrapper function `public.f_unaccent(text)`:
   - Body MUST be `SELECT public.unaccent('public.unaccent', $1)` — the schema-qualified dictionary form
   - NOT `SELECT unaccent($1)` — the simple form is not truly IMMUTABLE
   - Mark as `LANGUAGE sql IMMUTABLE PARALLEL SAFE STRICT`
   - Use `CREATE OR REPLACE FUNCTION`

3. Create RPC `public.search_venues_fuzzy(p_city_id uuid, p_query text, p_threshold float DEFAULT 0.15, p_limit int DEFAULT 200)`:
   - Returns `SETOF public.venues`
   - Language: `sql STABLE`
   - WHERE clause: `v.city_id = p_city_id AND (word_similarity(p_query, f_unaccent(lower(v.name_en))) > p_threshold OR f_unaccent(lower(v.name_en)) ILIKE '%' || p_query || '%' OR replace(f_unaccent(lower(v.name_en)), ' ', '') ILIKE '%' || p_query || '%')`
   - ORDER BY: `word_similarity(p_query, f_unaccent(lower(v.name_en))) DESC`
   - LIMIT: `p_limit`
   - CRITICAL: The expression `f_unaccent(lower(v.name_en))` must match the index expression exactly — `f_unaccent` wrapping `lower`, not the other way around

**File 2: `supabase/migrations/20260312_fuzzy_search_index.sql`** (non-transactional — CONCURRENTLY)

1. Add a comment at the top: "IMPORTANT: Run this OUTSIDE a transaction block. In psql use autocommit. Or run in Supabase SQL editor."
2. Create GIN trigram index:
   ```sql
   CREATE INDEX CONCURRENTLY IF NOT EXISTS venues_name_en_fuzzy
     ON public.venues
     USING GIN (f_unaccent(lower(name_en)) gin_trgm_ops);
   ```
3. The expression `f_unaccent(lower(name_en))` must match the RPC WHERE clause exactly

Use the exact SQL from the research document's Code Examples section. Do not deviate from the verified patterns.
